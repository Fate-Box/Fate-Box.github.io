このサイトについて

このおみくじサイトは、JavaScriptやCSSを使った動的なUI設計の練習として開発されました。APIを使用してリアルタイムでおみくじの結果を表示するシンプルかつ楽しい体験を目指しています。

開発の背景

「おみくじ」という日本の伝統文化をデジタル化することで、世界中の人々にその魅力を伝えたいという思いから、このサイトを作成しました。APIを通じたランダム要素やデザインに重点を置いています。

技術スタック

  • フロントエンド: HTML, CSS, JavaScript (Axiosを利用)
  • バックエンド: FastAPI (おみくじAPI)
  • デプロイ: GitHub Pages（フロントエンド）、Render（API）
